# BombermanUE4

Hello guys, I had great fun on this project! I hope you enjoy playing it!

Keys: (Also can be seen on the main menu map)

Player 1

Up: W
Down: S
Left: A
Right: D
Drop bomb/activate remote bomb: C

Player 2

Up: Up Arrow
Down: Down Arrow
Left: Left Arrow
Right: Right Arrow
Drop bomb/activate remote bomb: R-Shift
